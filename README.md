

## Description

This application is for secure election vote, we have used to encrypt the voting result [openPGP](https://www.npmjs.com/package/openpgp) javascript library 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode backend 
$ npm run start:dev

# runing frontend
$ cd client
$ npm start


```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
